## **한계점 & 향후 개선 방향**

1. work page에 한 게임만 연동되어서 아쉬움

2. 기존 프로젝트를 참고한 페이지
3. 로그인 후, 게임을 진행하고 게임 결과가 저장되면 좋을 듯 (→ 기록 깨기 or 순위 확인 가능하도록)
